C#数组

###创建

  1. 一维数组
    • int[] n =new int[10]; //不赋值
    • int[] n = {1,2,3}; //赋值
  2. 二维数组
    • int[,] = new int[2,3];
  3. 交错数组
    • int[][] = new int[2][];
    • int[][] = {new int[]{1,2},new int[]{1,2,3}};
  4. 参数数组（params）

###方法
Array 类是 C# 中所有数组的基类，它是在 System 命名空间中定义。Array 类提供了各种用于数组的属性和方法。

方法 描述
Reverse() 逆转数组
Sort() 排序数组
